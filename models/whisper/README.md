# Whisper Models

This directory stores Faster-Whisper model files.

Models are downloaded automatically on first run.

## Available models:
- tiny (~75MB) - fastest, lowest accuracy
- base (~150MB) 
- small (~500MB) - recommended
- medium (~1.5GB)
- large (~3GB) - best accuracy, requires more RAM

## For Jetson Orin Nano 8GB:
Recommended: `small` with `float16` compute type
