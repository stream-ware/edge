# Piper TTS Models

This directory stores Piper TTS voice models.

## Download Polish voice:

```bash
./scripts/download_piper_pl.sh medium gosia
```

## Available Polish voices:
- gosia (female) - recommended
- darkman (male)

## Quality levels:
- low (~15MB) - fastest
- medium (~60MB) - good balance
- high (~100MB) - best quality

## Files needed:
- `pl_PL-gosia-medium.onnx` - ONNX model
- `pl_PL-gosia-medium.onnx.json` - Config file
